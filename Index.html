WORK EXPERIENCE

I started working for TCS Ltd after graduating in 2004 in Computer Science Engineering from Madras university. I spent my time on the team developing Mainframe programs for the bank BBNT involving COBOL, JCL, DB2 and CICS. I used Expeditor to test the code and EXPLAIN to analyze DB2 SQL performance.

After two years I moved to JPMC bank project within TCS. I continued to work in mainframe environment where I developed several batch applications for Customer Information Systems. I also helped to migrate legacy IMS data to relational DB2 systems.

After a year, I was given a team to lead a subset of Customer Information Systems offshore team where I helped the team learn about the new DB2 system and parallely design new maintenance request that needs to be developed on the existing code.

At this point I got an opportunity to travel and work at the client location in Chicago. I spent almost a decade in designing several applications for Chase’ Customer information systems. I helped migrate many Mainframe-legacy programs to distributed Java applications.

In 2018, I returned to the same location where I started my career, but I knew I had to be relavent to the current tech trends. So, I started to pursue my learning in Analytics in which I trained in Tableau and got assigned to work on many “Proof of Concepts”. The two main criteria for the POCs are to prove that a successful connection can be established between in-premise cloud Data servers and Tableau and how data can be extracted from unstructured data using Python. I acquired the Tableau desktop specialist certificate in 2020 and using that knowledge I helped TCS acquire many projects in analytics using Tableau.

At the start of 2021 I decided I needed something different and found an opportunity to get certified as professional Data engineer from Google cloud. Using my cloud knowledge, I designed systems to capture streaming application data, store them in an in-house cloud repository and use the repository to produce metric based graphs in Grafana.
